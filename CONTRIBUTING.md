## Project Scope

This project is focused on providing utilities for the Fantacovid-19 application. We aim to maintain a high-quality codebase that is easy to understand and contribute to. Contributions can include bug fixes, new features, documentation improvements, and more.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct.

- **Be Respectful** and considerate to others
- **Be Collaborative** and open to feedback
- **Be Patient** and understanding of different perspectives
- **Be Inclusive** and welcoming to all contributors

## Getting Started

### Prerequisites

- Node.js (version 22 or higher)
- npm (version 11 or higher) or pnpm (version 10 or higher)
- Git
- A code editor (e.g., Visual Studio Code)

### Quick Start

1. Fork the repository and clone it to your local machine.
2. Install dependencies using `npm install` or `pnpm install`.
3. Create a new branch for your feature or bug fix.
4. Make your changes and commit them with clear and descriptive messages.
5. Push your changes to your forked repository.
6. Open a pull request against the main repository, describing your changes and why they are necessary
7. Wait for feedback and be responsive to any comments or suggestions.
8. Once your pull request is approved, it will be merged into the main branch.
9. Celebrate your contribution to the project!
